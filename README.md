# GL.Cache.ExpireTime
生成固定时间（缓存过期时间）（可用于多机器部署时的缓存不一致问题）
1. 内部没有没有任何特殊写法或者类库，所以可支持多版本，自行下载或拷贝代码使用都行
2. 支持指定 小时/分钟/秒 三种类型的任意过期间隔，支持返回两种类型的时间，一种是距今多少间隔，一种是具体的年月日时分秒
